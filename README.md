🎓 Student Management System

📌 Project Description

The Student Management System is a simple Python-based application used to manage student records.
It allows users to add, view, search, update, and delete student details easily.

This project is created for learning purposes and to understand basic Python concepts like:

- Lists
- Dictionaries
- Functions
- Loops
- Conditional statements

---

🚀 Features

- ➕ Add new student
- 📋 View all students
- 🔍 Search student by ID
- ✏️ Update student details
- ❌ Delete student record
- 💾 Data stored during program execution

---

🛠️ Technologies Used

- Python 3
- Command Line Interface (CLI)

---

📂 How to Run the Project

1. Make sure Python is installed.
2. Download or clone the repository.
3. Open terminal in project folder.
4. Run:

python main.py
